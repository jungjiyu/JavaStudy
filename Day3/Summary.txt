스캐너 > 화이트 스페이스를 기준으로 값을 구분하여 입력받는다

여러가지 자료형의 값을 입력받을 수 있다
	next어쩌구
		: 특수한얘 >> nextLine( ) : next어쩌구 계열에서 유일하게 엔터를 흡수할 수 있고, 중간 중간에 엔터가 아닌화이트 스페이스를 입력해도 이를 값의 구분기준으로 인식하지 않고 오직 엔터만을 기준으로 구분하여 한줄단위로 문장을 입력받을 수 있다
		: 평범한 얘들 >> 엔터를 사용해서 값을 구분할 순 있어도 엔터를 흡수하진 못한다
						: 주의해야할건 앞서 엔터를 여러번 사용해서 값을 구분하여 입력 받았을지라도 엔터가 남아있다면 항상 딱 하나만 남아있는다 
            : 거의 모든 자료형의 값을 읽어올 수 있는데 문자형은 이거 말고 문자열 입력받고 추가적으로 charAt(인덱스) 를 사용해야된다
                : scanner.nextLine().charAt(인덱스)  혹은  scanner.next().charAt(인덱스)  로 받는다
			1. next( ) : 문자열을 화이트 스페이스를 기준으로 입력받는다
			2. nextInt( ): 정수를 화이트 스페이스를 기준으로 입력받는다
      3. nextDouble( ) : 실수를 화이트 스페이스를 기준으로 입력받느다
      4. 이외에도 문자 빼곤 다양한 자료형의 값을 입력받을 수 있다
